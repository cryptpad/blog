### Scalable Server

This project is a complete rewrite of the CryptPad backend. It will allow the service to operate across multiple physical or logical nodes (for example virtual machines) to distribute ressources and avoid performance bottlenecks.

The work is already well underway. In the last few months performance issues have regularly affected users on our flagship instance (cryptpad.fr), to our knowledge by far the largest in terms of users and activity. While CryptPad performs very well on smaller instances, we have reached a ceiling for the single-node architecture that is hindering the project's growth.

We thank everyone for bearing with us, and are very happy to receive support to complete this work as fast as possible so that it can reach production in the next few months!

### CryptPad Notes

This project will replace Rich Text, one of the most used applications in CryptPad, with a much more contemporary alternative. The Rich Text application relies on [CKEditor 4](https://ckeditor.com/ckeditor-4/), a library that reached end-of-life in July 2023. It does not receive further updates or fixes and the new version's license is not open-source. Meanwhile, the Document application has started to fulfill many of the more office-like functions that Rich Text also offers. When considering a replacement we therefore leaned towards a more lightweight and screen-based solution suited for quick collaborative note taking and publishing on screens. 

After carefully studying our options we opted for [BlockNote](https://www.blocknotejs.org/), a simple yet powerful block-based editor that offers similar fomratting and features to Rich Text but through a much simplified interface using "`/`" commands. We're looking forward to updating the suite with this long awaited alternative!
